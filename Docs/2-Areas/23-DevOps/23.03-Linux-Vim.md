# Linux - Vim Cheatsheet

## 자주 쓰는 명령어

### 파일 실행

| 명령어 | 설명 |
|--------|------|
| `vi file` | 파일 열기 |
| `vi file1 file2` | 파일 1, 2 차례로 열기 |
| `view file` / `vi -R file` | 읽기 전용으로 열기 |
| `vi + file` | 커서를 마지막 행에 위치하여 열기 |
| `vi +n file` | 커서를 n행에 위치하여 열기 |
| `vi -r file` | 손상된 파일 회복 |

### 입력 모드 전환

| 명령어 | 설명 |
|--------|------|
| `i` | 커서 위치에서 입력 모드 |
| `I` | 행 맨 앞에서 입력 모드 |
| `a` | 커서 다음 위치에서 입력 모드 |
| `A` | 행 맨 끝에서 입력 모드 |
| `o` | 현재 행 아래에 빈 줄 삽입 후 입력 모드 |
| `O` | 현재 행 위에 빈 줄 삽입 후 입력 모드 |
| `R` | 덮어쓰기 모드 |
| `s` | 커서 위치 한 글자 삭제 후 입력 모드 |
| `cc` | 현재 행 전체 삭제 후 입력 모드 |
| `Esc` | 명령 모드로 복귀 |

### 저장 및 종료

| 명령어 | 설명 |
|--------|------|
| `:w` | 저장 |
| `:w file` | 다른 이름으로 저장 |
| `:w >> file` | 파일에 덧붙여 저장 |
| `:q` | 종료 |
| `:q!` | 강제 종료 (변경 사항 버림) |
| `:wq!` | 강제 저장 후 종료 |
| `ZZ` | 저장 후 종료 |
| `:e file` | 다른 파일 불러오기 |
| `:e#` | 직전에 열었던 파일로 전환 |

### 문자열 찾기 / 대체

| 명령어 | 설명 |
|--------|------|
| `/name` | `name` 문자열 검색 (정방향) |
| `n` | 다음 검색 결과로 이동 |
| `N` | 이전 검색 결과로 이동 |
| `:s/str/rep` | 현재 행의 첫 번째 `str`을 `rep`로 대체 |
| `:1,.s/str/rep/` | 1행부터 현재 행까지 대체 |
| `:%s/str/rep/g` | 파일 전체 `str`을 `rep`로 전부 대체 |
| `:.,$s/aaa/bbb` | 커서 위치부터 파일 끝까지 `aaa`를 `bbb`로 대체 |

## 자주 하는 실수
- 저장하지 않고 `:q` 하면 `E37: No write since last change` 에러 → `:q!` 로 강제 종료
- 입력 모드에서 `Esc` 없이 명령어 입력하면 텍스트로 삽입됨
