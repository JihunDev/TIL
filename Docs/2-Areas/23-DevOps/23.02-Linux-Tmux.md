# Linux - Tmux Cheatsheet

## 설치

```bash
# Ubuntu / Debian
sudo apt install tmux

# macOS
brew install tmux
```

## 자주 쓰는 명령어

### 세션 관리

| 명령어 | 설명 |
|--------|------|
| `tmux` | 새 세션 시작 |
| `tmux new -s name` | 이름 지정 세션 시작 |
| `tmux ls` | 세션 목록 확인 |
| `tmux attach -t name` | 세션 재접속 |
| `Ctrl + b d` | tmux 종료 (백그라운드 유지) |

### 창(Window) 관리

| 명령어 | 설명 |
|--------|------|
| `Ctrl + b c` | 새 창 생성 |
| `Ctrl + b q` | 창 번호 확인 |
| `Ctrl + b d` | 현재 세션 분리 (백그라운드 유지) |
| `Ctrl + d` | 현재 창 닫기 |

### 분할(Pane) 관리

| 명령어 | 설명 |
|--------|------|
| `Ctrl + b "` | 가로 분할 (위/아래) |
| `Ctrl + b %` | 세로 분할 (좌/우) |
| `Ctrl + b 방향키` | 방향키 위치 창으로 이동 |

## 자주 하는 실수
- `Ctrl + b` 먼저 누르고 **손을 뗀 뒤** 다음 키 입력 (`Ctrl + b` → `"`)
- 세션 재접속 없이 터미널 닫으면 백그라운드에 세션이 남아 있음 (`tmux ls`로 확인)
