# Signal - ICP 가속도계 센서

## 개요
- **핵심 개념**: 진동, 충격, 가속도를 측정하는 압전(Piezoelectric) 기반 센서. ICP(IEPE/CCLD)는 내장 증폭 회로를 갖춘 Voltage Type의 표준 인터페이스
- **활용 주파수 범위**: ICP Type → 중~고주파 / Capacitance Type → 저주파

## 상세 분석

### ICP Type (고주파 진동/충격 측정)

**활용**: 진동 측정, 사운드 측정, 충격 측정, 가속도 측정 — 보통~높은 주파수 분석

#### Charge Type (전하형)

| 특징 | 내용 |
|------|------|
| 외부 전원 | 불필요 |
| 응답성 | 빠름 |
| 단점 | 별도 전하 증폭기(Charge Amplifier) 필요 |

**연결 구성**: `Charge 가속도계 → In-Line Charge Converter → ICP 시그널 컨디셔너(앰프)`

#### Voltage Type (전압형, ICP/IEPE/CCLD)

| 특징 | 내용 |
|------|------|
| 외부 전원 | 필요 (센서 내 증폭 회로 구동) |
| 안정화 시간 | 필요 (전원 인가 후 안정화 대기) |
| 사용 편의성 | Charge Type보다 간단 |

**연결 구성**: `ICP 가속도계 → ICP 시그널 컨디셔너(앰프)`

### Capacitance Type (저주파 측정)

**활용**: 지진 측정, 지진파 분석 등 저주파 측정에 적합

### Strain Gauge Type

압력/힘 측정 응용

---

### 내부 구조 비교 (압전 소자 배치 방식)

| 구조 | 특징 | 장점 | 단점 |
|------|------|------|------|
| **Shear (전단형)** | 압전 소자가 전단 응력 감지 | 온도/베이스 응력 영향 적음, 우수한 공진 특성, 내구성 좋음 | — |
| **Flexural (굴곡형)** | 압전 소자가 굽힘 응력 감지 | 가격 저렴, 단순 구조 | 낮은 공진 주파수, 구조 취약 |
| **Compression (압축형)** | 압전 소자가 압축 응력 감지 | 탁월한 공진 특성, 내구성 좋음 | 온도/베이스 응력 변화에 민감 |

## 엣지 케이스 / 잠재적 버그
- [ ] Voltage Type 측정 시 전원 인가 직후 안정화 시간 전 데이터는 유효하지 않음
- [ ] 케이블 움직임이 노이즈로 유입될 수 있음 (Triboelectric 효과) → 고정 필수

## 실무 적용 포인트
- [ ] FODRo 모터/구동부 진동 모니터링: Shear Type ICP 가속도계 + DAQ → RMS 기반 이상 감지
- [ ] 측정 전 센서 마운팅 방법 확인 (나사 고정 > 왁스 > 자석 순으로 고주파 특성 우수)
