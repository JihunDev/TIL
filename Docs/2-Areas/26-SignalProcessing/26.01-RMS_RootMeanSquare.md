# Signal - RMS (Root Mean Square)

## 핵심 요점
> RMS는 시간에 따라 변하는 신호(진동, 전압 등)의 **실효값**을 나타내는 지표. 신호의 에너지 크기를 단일 값으로 표현할 때 사용.

## 내용

### 정의

임의의 n개의 값 x₁, x₂, ..., xₙ에 대한 제곱평균제곱근:

$$RMS = \sqrt{\frac{1}{n}\sum_{i=1}^{n} x_i^2}$$

- 연속 신호: $RMS = \sqrt{\frac{1}{T}\int_0^T x(t)^2 \, dt}$

### 주요 사용처

| 분야 | 의미 |
|------|------|
| 교류 전압(AC) | 실효 전압값 (220V = RMS 값) |
| 진동 측정 | 진동 강도의 크기 평가 |
| 음향 | 음압 레벨(SPL) 계산 |
| 신호 분석 | 신호 에너지(파워) 측정 |

### 주의사항 / 함정
- 피크값(Peak)과 혼동 주의: RMS ≠ 평균값, RMS = Peak / √2 (순수 정현파의 경우)
- DC 성분이 포함된 신호에서는 RMS와 평균값이 다름
