# HW - DC/DC Converter

## 개요
- **문제 정의**: 반도체 부품마다 고유한 동작전압 범위가 다르고, 불안정한 전원 공급 시 오작동 및 열화 발생
- **핵심 개념**: 직류 전압을 원하는 레벨로 변환하는 전원 장치
- **관련 기술 비교**: 리니어 레귤레이터 vs 스위칭 레귤레이터

## 상세 분석

### 종류 (변환 방향)

| 이름 | 동작 |
|------|------|
| 강압 (Buck / Step-Down) | 전압을 낮추는 전원 장치 |
| 승압 (Boost / Step-Up) | 전압을 높이는 전원 장치 |
| 승강압 (Buck-Boost) | 전압을 높이고 낮추는 전원 장치 |
| 부전압 / 반전 (Inverting) | 마이너스 전압을 생성하는 전원 장치 |

### 리니어 레귤레이터 (Linear Regulator)

- 입력-출력 관계가 선형 동작 (시리즈 레귤레이터라고도 함)
- 입력-출력 사이에 제어 소자를 직렬로 배치하여 강압

| 장점 | 단점 |
|------|------|
| 심플한 회로 | 낮은 효율 |
| 적은 외장 부품 | 큰 발열 (전압차 × 전류만큼 열 소비) |
| 노이즈 적음 | 강압 변환만 가능 |

**적합한 용도**: 소전력, 노이즈에 민감한 아날로그 회로

### 스위칭 레귤레이터 (Switching Regulator)

- MOSFET 스위치 소자를 ON/OFF 반복하며 출력 전압 조절
- 출력이 목표값에 도달하면 스위치 OFF → 입력 전력 소비 최소화

| 장점 | 단점 |
|------|------|
| 우수한 효율 (80~95%) | 많은 외장 부품 (인덕터, 커패시터) |
| 적은 발열 | 설계가 어려움 |
| 승압 / 강압 / 부전압 모두 가능 | 스위칭 노이즈 발생 |

**적합한 용도**: 배터리 구동 시스템, 고전류 부하

### 선택 기준 요약

| 조건 | 추천 |
|------|------|
| 노이즈 민감 (오디오, 센서) | 리니어 레귤레이터 |
| 효율 중요 (배터리, 모바일) | 스위칭 레귤레이터 |
| 승압 필요 | 스위칭 레귤레이터 |
| 전압차 작고 저전류 | 리니어 레귤레이터 |

## 엣지 케이스 / 잠재적 버그
- [ ] 스위칭 레귤레이터의 노이즈가 인근 아날로그 센서에 간섭 가능
- [ ] 리니어 레귤레이터에서 입출력 전압차가 크면 발열 심각 → 방열판 필수
