# HW - Vcc / Vee / Vdd / Vss

## 개요
- **핵심 개념**: 전자회로에서 전원 핀 명칭은 BJT(트랜지스터)와 FET(전계효과 트랜지스터)의 단자 이름에서 유래
- **선수 지식**: BJT, FET, CMOS 기초

## 상세 분석

### 정의

| 명칭 | 풀네임 | 원래 단자 |
|------|--------|----------|
| Vcc | Voltage @ Collector (= Voltage of Common Collector) | BJT Collector |
| Vee | Voltage @ Emitter | BJT Emitter |
| Vdd | Voltage @ Drain | FET Drain |
| Vss | Voltage @ Source | FET Source |

### BJT / FET 대응 관계

| BJT | FET | 의미 |
|-----|-----|------|
| Vcc (C) | Vdd (D) | + 입력전압 |
| Vee (E) | Vss (S) | - 입력전압 (GND) |

> Vss = Ground, 접지, 일반적으로 0V

### 실제 회로에서의 사용
- **디지털 IC (CMOS)**: Vdd (+전원) / Vss (GND) 명칭 주로 사용
- **아날로그 BJT 회로**: Vcc (+전원) / Vee (GND 또는 부전원) 명칭 사용
- 혼용되는 경우가 많으므로 데이터시트에서 반드시 확인 필요

## 엣지 케이스 / 잠재적 버그
- [ ] 데이터시트에서 Vcc와 Vdd가 혼용되는 경우가 있으므로 맹신 금지
- [ ] 음(−) 전원이 필요한 OP-AMP 회로에서 Vee가 GND가 아닌 부전압일 수 있음

## 실무 적용 포인트
- [ ] 회로 설계 시 파트 데이터시트의 absolute maximum ratings 에서 Vcc/Vdd 허용 범위 확인
- [ ] MCU와 센서 연결 시 동작 전압 레벨 (3.3V vs 5V) 반드시 매칭
