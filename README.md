## Using Tool
- [Netlify](https://www.netlify.com/)
- [Docsify](https://docsify.js.org/)

## Docsify
### Install
```shell
$ npm i docsify-cli -g
```
### 초기 설정
```shell
$ docsify init ./docs
```
### 실행 방법
```shell
$ docsify serve ./docs
```
